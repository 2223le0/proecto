Reglas

Estas seran las reglas utilizadas para el concurso de programacion 2024

Costo

El costo por la inscripción del equipo será de 50 pesos.

classroom

Para la revisión de la página y del código se utilizará la herramienta classroom.

Equipos

El equipo debera conformarse por máximo 5 integrantes.

Visual studio

Deberán utilizar visual studio code como herramienta de programación.

Autoria

El equipo deberá presentar la página de su autoría.

Tipo de pagina

La página deberá ser informativa.
